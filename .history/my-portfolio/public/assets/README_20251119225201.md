# Assets Directory

## Avatar Image

Place your profile photo here as `avatar.jpg` (or `.png`, `.webp`).

**Recommended specifications:**
- **Format:** JPG, PNG, or WebP
- **Dimensions:** 400x400px minimum (square)
- **File size:** < 200KB (optimize for web)
- **Subject:** Professional headshot with clear face visibility

**Optimization tips:**
- Use tools like TinyPNG or Squoosh to compress
- Ensure good lighting and contrast
- Center your face in the frame

**Update path in Home.jsx:**
```jsx
avatarSrc: "/assets/avatar.jpg"
```

Replace with your actual filename if different.
